---
name: Conserver les emojis dans le README
description: L'utilisateur souhaite garder les emojis dans le README pour apporter de la fraîcheur visuelle
type: feedback
---

Ne pas supprimer les emojis du README lors de réécriture ou reformatage. L'utilisateur les considère comme un élément de style important.

**Why:** L'utilisateur a explicitement reproché leur suppression lors d'une réécriture du README.

**How to apply:** Lors de toute modification du README, conserver les emojis décoratifs présents dans le document original.
